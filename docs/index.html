Projects

Selected work and stacks

Operating System Project

C, Concurrency, File Systems

This project extends the Pintos OS skeleton with three major subsystems: user programs (processes & syscalls), thread scheduling and synchronization, and a file system redesign (FFS-style layout + Clock/Second-Chance buffer cache).

Relational Database Engine

Java, B+Tree, ARIES Recovery

This project implements a relational database engine featuring clustered B+Tree indexing, a query engine supporting multiple join algorithms, multi-granularity locking for safe concurrent access, and ARIES-style logging with checkpointing to ensure transactional durability and correctness.

Unix-like shell Project

Operating Systems, Shell

Built a Unix-like mini shell that executes built-in commands (ls, pwd) and external programs, with a tokenizer-driven parser supporting I/O redirection (<, >), pipelines (|), and job control via Ctrl-C/Ctrl-Z, demonstrating process management, file-descriptor manipulation, and signal handling using core OS principles.

My World Game

Java, OOP, Data Structures, A* Algorithm, GUI

The game features a player character that can move around a grid-based world, collect items, and avoid enemies based on basic data structures, algorithms, and software engineering principles.

2-Stage Pipelined RISC-V CPU

Logisim, CPU Microarchitecture, 2-Stage Pipeline, RISC-V

Designed and implemented a 2-stage (Fetch / Execute) pipelined CPU supporting a subset of the RISC-V ISA using Logisim. Built the full datapath and control logic, handled pipeline hazards, and verified cycle-accurate behavior using provided test programs.
